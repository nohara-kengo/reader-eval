# デプロイ・インフラ運用 Runbook（運用者向け）

> **読者と位置づけ**: 本書は**運用者向け**の参考情報です。**実装（および Claude / Codex 等の AI コーディング）の主たる関心外**です。
> `main` / `dev` への push で**自動デプロイ**されるため、アプリ実装でデプロイ手順を意識する必要はありません。
> AI に渡す情報は要件定義書の方針どおり「**Docker で動く（ローカルは `docker compose`）**」に留め、ここに書く具体（VM 構築・公開・デプロイ基盤）は **docker-compose を書くなど必要なときに、必要な範囲だけ** 参照・共有してください（ハルシネーション・ノイズ防止）。

---

## 1. 稼働環境（社内IDC / オンプレミス）

- **設置**: 本社近傍のデータセンター（社内で「IDC」と呼称）のラックに設置済みの物理マシン（Windows Server、複数台稼働中）。新規調達不要。
- **基盤**: 物理マシン上の **Hyper-V** に **Ubuntu（最新LTS）VM** を構築し、その上で **Docker** でアプリを運用。
- **VM 割当スペック（目安）**:

  | リソース | 割当 | 備考 |
  |---|---|---|
  | CPU | 4 Core | 大抵のアプリはこれで満足に動作 |
  | RAM | 16GB〜32GB | まずは様子見。必要に応じ増減 |
  | ストレージ | 〜150GB | DB・コンテナイメージ・出力ファイル等を含む |

- **アプリ実行**: すべて Docker コンテナで常駐（アプリ（Next.js: UI＋API）/ DB / リバースプロキシ）。
- **接続性（社内LAN）**: 構築した Ubuntu VM へは社内LANから特別な設定なしで SSH 接続が可能。

## 2. コンテナ管理 / デプロイ基盤（Coolify）

- **Coolify**（OSS のセルフホスティング PaaS）でデプロイ・リバースプロキシ・ログ・GitHub 連携・CI/CD を一元管理。
- 一度導入すれば細かい設定を意識せず運用でき、その後の管理が容易。
- OS / Docker / Coolify の更新は自社運用（クラウドと異なりパッチ責任は自社）。定期メンテナンス枠を設ける。

## 3. 公開・ドメイン

- **外部公開（必要時）**: **Cloudflare Tunnel** で特定ポートのみ Cloudflare 経由で公開。VM のポートを直接インターネットへ晒さない。
- **ドメイン**: `*.comthink.co.jp` または `*.dev-nct.jp` を社内ですぐ提供可能。本番／開発でサブドメインを割り当てる。

## 4. CI/CD（自動デプロイ）

- **GitHub** の `main` → 本番 / `dev` → 開発 へ、**push をトリガに自動デプロイ**。
- ビルド（Docker イメージ化）→ デプロイ → リバースプロキシ反映までを基盤側に集約し、手動オペレーションを最小化。
- CI（lint / typecheck / test / build, e2e 等）は GitHub Actions（リポジトリ `.github/workflows/`）。

## 5. 構築方針（WSL検証 → コマンドベース → 自動化）

- 実機（本番VM）へ直接手作業で構築せず、まず**手元の WSL（Ubuntu）で同等環境を再現して検証**する。
- VM 初期セットアップ（Docker / Coolify 導入、パッケージ、ファイアウォール、Cloudflare Tunnel 設定 等）は**シェルスクリプト等で手順化**し、属人化を避ける。
- アプリ構成は `docker compose` / 基盤の定義ファイルで宣言的に管理し Git で版管理する。

## 6. バックアップ

- ①DB の定期ダンプ ＋ ②Hyper-V の **VM スナップショット** を組合せ復旧可能性を担保。
- 保存先（別ストレージ／別マシン）と頻度は運用で定める。データ所在は社内IDCに閉じる。

## 7. コスト（追加ランニング実質ゼロ）

既存設備・OSS・無料枠中心のため、本件のための追加ランニングコストは実質ゼロ。

| 項目 | 費用 | 備考 |
|---|---|---|
| 物理マシン / ラック / 電気代 | 既存固定費 | 稼働中設備を流用 |
| Hyper-V / Windows Server | 既存ライセンス | 稼働中ホスト |
| Ubuntu VM / Docker / Coolify | ¥0 | OSS・無償 |
| Cloudflare Tunnel | ¥0 | 無料枠 |
| ドメイン / GitHub / CI/CD | 既存 | 流用 |
